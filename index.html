Projects

Procedural Voxel Terrain Generation in UE5 C++ and PCG

I built a fully procedural voxel terrain system from scratch in UE5 using C++ and the PCG plugin as part of mining simulator game that I am solo-developing. Chunks generate asynchronously at runtime using a single PCG graph instance, with the voxel data displayed through a custom chunked mesh system. Rather than a generic auto-material, the terrain material is driven directly by the voxel data itself — each geology type (grass, dirt, rock, coal) maps to its own texture layer via a runtime weight map texture written from C++.
